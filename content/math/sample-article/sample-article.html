3.2 位相群，位相環

adele環，idele群を構成し，位相の入れ方を解説する．位相空間論，位相群，位相環に関してわからないことは§3に必要であろう事項をまとめたので，そちらを参照されたい．また以下の記法を採用する．

  • Fを代数体とする．
  • 位相空間Xの位相1 をOXと書き，位相空間(X,OX)からその部分集合Y に入る相対位相はOX ∩ Y と書く．2
  • Fの有限素点全体をP<∞，無限素点全体をP∞，P = P<∞∪ P∞とおく．p ∈ Pに対してFpを，Fをpで完備化した位相体とする．p ∈ P<∞に対してFpの付値環をJpとする．

1 adele環

定義 1.adele環

  1. P∞⊆ S ⋐ P なるSに対して，位相環F𝔸(S)を位相環の直積F𝔸(S) = ∏ ⁡ p∈SFp ×∏ p∈P∖SJpで定める．3つまり演算は各成分ごとの演算であり，位相は直積位相OF𝔸(S)である．

  2. Fのadele環F𝔸を

    F𝔸 = ⋃ P∞⊆S⋐P F𝔸(S) = {(xp) ∈∏ p∈P Fp |#{p ∈ P∣xp∉Jp} < ∞ }

    と定める．演算は成分ごとの演算で定める．

  3. F𝔸の位相OF𝔸で，

    1. (F𝔸,OF𝔸)が位相環
    2. F𝔸(P∞)が開
    3. F𝔸(P∞)に入る相対位相とF𝔸(P∞)の位相が一致

    を満たすようなものが一意的に存在する．この位相によりF𝔸に位相を入れ位相環とする．

演算について，(xp),(yp) ∈ F𝔸に対して，xp + yp,−xp,xpyp∉Jp⟹xp∉Jpまたはyp ∈ Jpゆえ，F𝔸は演算で閉じており確かに環になっている．位相は∏ ⁡ p∈P Fpから入る相対位相よりも真に強い位相が入れられている4 （命題6を参照）．以下の命題で挙げるF𝔸の位相OF𝔸の五つの特徴づけが理解の助けになるだろう．挙げる特徴づけについて，O1,O2は位相環として位相が分かりやすい．O3,O4は位相空間として位相が分かりやすく具体的に扱いやすい．O5は圏論的な特徴づけで簡潔である．

命題 2.

以下のF𝔸の位相O1(= OF𝔸),O2,O3,O4 ⊆ 2F𝔸は一意的に存在し，すべて一致する．

  1. O1

    1. (F𝔸,O1)が位相環
    2. F𝔸(P∞)が開
    3. F𝔸(P∞)に入る相対位相と直積位相が一致

  2. O2

    1. 任意のP∞⊆ S ⋐ PでF𝔸(S)が開
    2. 任意のP∞⊆ S ⋐ PでF𝔸(S)に入る相対位相と直積位相が一致
  3. O3
    B3 = {∏ ⁡ p∈SUp ×∏ p∈P∖SJp∣P∞⊆ S ⋐ P,∀ ⁡p ∈ S,Up ⊆ Fpは開}を開基にもつ
  4. O4
    B4 = ⋃ ⁡ P∞⊆S⋐P {U ⊆ F𝔸(S)∣F𝔸(S)で開}を開基にもつ
  5. O5
    任意のP∞⊆ S ⋐ Pに対して自然な包含写像iS : F𝔸(S) → F𝔸が連続になるような最も強い位相

注意 3. O1以外には位相環としての，つまり演算の連続性に関する条件がないが，それは条件を満たせば演算が連続になるほど，位相についての強い条件だからである．

注意 4 (圏論的な見方). O5の条件は，圏論的にはO5を備えたF𝔸が{F𝔸(S)∣P∞⊆ S ⋐ P}の極限であるという意味である．P∞⊆ S ⊆ T ⋐ Pのとき，包含写像F𝔸(S) → F𝔸(T)は連続準同型であるので，この射によって{F𝔸(S)∣P∞⊆ S ⋐ P}は位相環の圏での図式をなす．O5の条件は，F𝔸がこの図式の余極限，すなわちF𝔸 = lim ⁡ S→F𝔸(S)であることと同値である．

補題 5.

可換な位相群(H,OH)がある可換群Gの部分群になっているとする．このとき以下を満たすGの位相OGが一意的に存在する．

  2. Hは開
  3. Hに入る相対位相とHの位相が一致

(g + U) ∩ (h + V ) = x + ((g − x) + U) ∩ ((h − x) + V )

はBの元であり，もちろん(g + U) ∩ (h + V )に包まれxを含む．よってBは開基である．Bが生成する位相をOGとおく．
2°次にOGが条件(1)-(3)を満たすことを示す．まず定義よりH ∈Bゆえ(2)は満たされている．(1)を示す．加法の連続性を示す．任意にg,h ∈ Gをとる．g + hを含む任意の開集合Wに対してg,hの近傍U,V であってU + V ⊆ Wなるものが存在すれば良い．1°と同様の議論より − (g + h) + Wは0(= 0 + 0)を含むHの開集合であり，Hが位相群であることから，あるHでの0の近傍U′,V ′が存在し，U′ + V ′⊆−(g + h) + Wとなる．U = g + U′,V = h + V とおけば良い．逆元の連続性も同様である．(3)を示す．定義よりOH ⊆OG ∩ Hである．OG ∩ H ⊆OHを示すためには{B ∩ H∣B ∈B}⊆OHを示せば十分．GがHによる剰余類の非交和であることを思い出せば，任意のg + U ∈Bについて，g ∈ Hなら(g + U) ⊆ Hゆえ(g + U) ∩ H = g + U ∈OH，g∉Hならg + U ∩ H = ∅ ∈OHである．

2 idele群

P∞⊆ S ⋐ P∞とする．F𝔸(S)× = ∏ ⁡ p∈SFp××∏ p∈P∖SJp×にはFp,Jpたちから標準的な位相OF𝔸(S)×が定まる．7 位相体Fpの乗法群Fp×は位相群であり，Jp×はその位相部分群ゆえ位相群である．位相群の直積ゆえ(F𝔸(S)×,OF𝔸(S)×)は位相群である．

定義 7.idele群

F𝔸の乗法群F𝔸×をFのidele群という．

F𝔸× = ⋃ P∞⊆S⋐P F𝔸(S)× = {(x p) ∈∏ p∈P Fp ×∣#{p ∈ P∣xp∉Jp×} < ∞ }

である． F𝔸×の位相OF𝔸×で

  1. (F𝔸×,OF𝔸×)が位相群
  2. F𝔸(P∞)×が開
  3. F𝔸(P∞)×に入る相対位相とF𝔸(P∞)×の位相が一致

を満たすようなものが一意的に存在する．この位相によりF𝔸×に位相を入れ位相群とする．

補題5よりOF𝔸×の存在と一意性は保証される．F𝔸は位相環だが位相体では（というか整域ですら）ないため，F𝔸×にF𝔸から相対位相を入れても乗法逆元をとる写像が連続となるとは限らず，位相群になるとは限らない．8 これを解消するためにF𝔸×には相対位相よりも真に強い位相が入っている（命題9を参照）．adele環の場合と同様にidele群の位相について以下のような特徴づけがある．

3.1 位相空間論

定義 10.位相の定義

Xを集合とする．以下を満たすO ⊆ 2XをXの位相という．Xとその位相一つの組み(X,O)を位相空間といい，Oの元を開集合という．

  1. ∅,X ∈O
  2. U1,U2 ∈O⟹U1 ∩ U2 ∈O
  3. ∀ ⁡λ ∈ Λ,Uλ ∈O⟹⋃ ⁡ λ∈ΛUλ ∈O

以下，(X,O)を位相空間とする．

定義 11.開基

集合族B ⊆Oが開基であるとは，任意のx ∈ Xとx ∈ U ∈Oに対して，あるB ∈Bが存在してx ∈ B ⊆ Uとなることである．Bが開基なら任意のU ∈OはBの元の和集合として書ける．

命題 12.開基の公理

  1. BをXの開基とする．このとき以下が成り立つ．

    1. ⋃ ⁡ B = X.
    2. ∀ ⁡B1,B2 ∈B,∀ ⁡x ∈ B1 ∩ B2,∃ ⁡C ∈B,x ∈ C ⊆ B1 ∩ B2.
  2. 単なる集合Xの部分集合族Bであって上記の条件を満たすものが与えられているとする．このときBを開基とするようなXの位相が一意的に存在する．すなわち適切に開基を定めることで位相を定義できる．

(2)は具体的には{⋃ ⁡ λ∈ΛBλ∣∀ ⁡λ ∈ Λ,Bλ ∈B}を開集合全体とする位相である．

定義 13.相対位相，直積位相

  1. Y ⊆ Xに対して

    O ∩ X = {U ∩ X∣U ∈O}

    はY の位相となる．これを(X,O)から入るY の相対位相という．これは包含写像Y → Xが連続となる最弱の位相である．

  2. 位相空間の族{(Xλ,Oλ∣λ ∈ Λ}が与えられているとする．

    B = {∏ λ∈ΛUλ |有限個のλを除いてUλ = Xλ }

    を開基とする位相が一意的に存在する．この位相を∏ ⁡ λ∈ΛXλの直積位相という．10これは全ての射影∏ ⁡ λ∈ΛXλ → Xλが連続となるような最弱の位相である．

命題 14.相対位相と直積位相の可換性

各λ ∈ Λに対して位相空間Xλとその部分空間Aλが与えられているとする．XλからAλに入れた相対位相たちの直積位相と，∏ ⁡ λ∈ΛXλに入る直積位相からの∏ ⁡ λ∈ΛAλに入れた相対位相は一致する．

定義 15.近傍，基本近傍系

  1. N ⊆ Xがx ∈ Xの近傍であるとは，あるU ∈Oが存在しx ∈ U ⊆ Nとなることである．
  2. N ⊆ 2Xがx ∈ Xの基本近傍系であるとは，xの任意の近傍Nに対してあるM ∈Nが存在しM ⊆ Nとなることである．

3.2 位相群，位相環

定義 16.位相群，位相環，位相体

  1. 位相を備えた群Gであって

    G × G ∋ (x,y)↦xy ∈ G,G ∋ x↦x−1 ∈ G

    が連続となるようなものを位相群という．ただしG × Gには直積位相を入れる．

  2. 位相を備えた環Rであって

    R × R ∋ (x,y)↦x + y ∈ R,R ∋ x↦ − x ∈ R,R × R ∋ (x,y)↦xy ∈ R

    が連続となるようなものを位相環という．ただしR × Rには直積位相を入れる．

  3. 位相を備えた体Fであって

    F × F ∋ (x,y)↦x + y ∈ F,F ∋ x↦ − x ∈ F, F × F ∋ (x,y)↦xy ∈ F,F ∋ x↦x−1 ∈ F

    が連続となるようなものを位相体という．ただしF × Fには直積位相を入れる．

以下，Gを位相群，Rを位相環，Fを位相体とする．

命題 17.演算は同相

任意のg ∈ Gについて，

Lg : G ∋ x↦gx ∈ G,Rg : G ∋ x↦xg ∈ G,Inv : G ∋ x↦x−1 ∈ G

とおく．これらは連続であり

Lg −1 = Lg−1,Rg−1 = R g−1,Inv−1 = Inv

ゆえ，これらは全て同相写像である．

注意 18. 位相環Rの演算Lr : R ∋ x↦rx ∈ R,Rr : R ∋ x↦xr ∈ R(r ∈ R)は同相写像とは限らない．

定義 19.位相部分群，部分位相環，部分位相体

位相群の部分集合であって，相対位相を備えた部分群でそれ自身が位相群であるものを，位相部分群という．11位相群の任意の部分群は相対位相により常に位相部分群になる．位相環，位相体についても同様である．

定義 20.位相群，位相環の直積

  1. 位相群の族{Gλ∣λ ∈ Λ}が与えられているとする．位相群の直積∏ ⁡ λ∈ΛGλに，演算を各成分ごとの演算で，位相を直積位相で定める．これは位相群となる．
  2. 位相環の族{Rλ∣λ ∈ Λ}が与えられているとする．位相群の直積∏ ⁡ λ∈ΛRλに，演算を各成分ごとの演算で，位相を直積位相で定める．これは位相環となる．

命題 21.位相群の0の基本近傍系

位相群(G,O)の単位元1の基本近傍系N0が与えられているとする．Gの位相O′がN0を0の基本近傍系として持ち，(G,O′)位相群となるならばO′ = Oである．

証明. (G,O)の各点x ∈ Gについて演算Lxが同相写像ゆえxの基本近傍系としてxN0が取れるため，{xN0∣x ∈ G}は(G,O)の基本近傍系全体となる．(G,O′)でも同様に各点x ∈ Gの基本近傍系としてxN0を取れるため，{xN0∣x ∈ G}が基本近傍系全体となる．同一の基本近傍系全体を持つためO′ = O． □
